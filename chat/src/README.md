# Chat en Vivo con Spring WebSocket y Thymeleaf

Este proyecto es una aplicación de chat en tiempo real que utiliza **Spring WebSocket** para la comunicación en tiempo real y **Thymeleaf** para la generación de vistas. Permite a los usuarios conectarse al chat, enviar mensajes y recibir notificaciones cuando otros usuarios ingresan o salen del chat.

## Objetivo

El objetivo de este proyecto es conocer el funcionamiento de WebSocket junto con los protocolos necesarios para su funcionamiento y la interacción entre los agentes que lo utilizan.

## Tecnologías Utilizadas

- **Java**: Lenguaje de programación principal.
- **Spring Boot**: Framework para construir la aplicación.
- **Spring WebSocket**: Para la comunicación en tiempo real.
- **Thymeleaf**: Motor de plantillas para la generación de vistas.
- **SockJS** y **STOMP**: Protocolos para la comunicación de WebSocket.

## Funcionalidades

- Conexión de usuarios al chat.
- Envío y recepción de mensajes en tiempo real.
- Notificaciones cuando un usuario entra o sale del chat.
